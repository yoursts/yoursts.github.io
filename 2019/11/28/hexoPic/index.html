hexo插入图片

2019-11-28
hexo

用markdown写hexo文章时，从本地上传图片，图片的地址用的便是本地图片的地址，启动服务后，图片不显示，根据地址找不到图片。

本来做法是想把图片复制过来，用相对地址，可如果图片过多，每个都要复制，还有从网上复制链接的图片，很麻烦。

网上有大佬写的插件 hexo-easy-images

Hexo 博客插件，post 中随意引用本地图片、远程图片，插件将自动处理图片 copy 与 download及引用路径，本地预览与线上都能完美显示图片。

安装

自定义配置

以下配置非必需，在 your-hexo-project/_config.yml 添加。

介绍

这个插件在编译时会将post中使用的图片，无论是本地还是网上的图片, copy 到 source/images/your_post_id 文件夹下，并将 post文件内图片引用路径改为本地相对路径，保证本地 Markdown 编辑器正确显示。

同时在生成 html 文件时，会将 html 中图片路径改为绝对路径。故，无论本地线上，图片都无需担心。

PS:

网上还有很多人使用另一个插件: hexo-asset-image

使用这个插件时，post_asset_folder:true 这个 _config.yml 文件里的参数需要设置成true，意思就是new 文章时会新建一个同名文件夹来存放该文章的图片。上传时会将文件夹一起上传，但是好像需要用官方提供的标签，如果用markdown写法好像也有点区别，所以我没有使用。

如果用hexo-easy-images这个插件，post_asset_folder:false 这个属性需要还回去，改成false。

PSS:

使用markdown免费编辑器Typora 在设置中有选项，可以直接将本地图片传入指定文件夹，还能够将其编译成url，也很方便，可以试试。
